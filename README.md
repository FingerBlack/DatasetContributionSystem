# DatasetContributionSystem

#### 任务目标

* 数据集的管理

* 用户的管理

* 用户上传与下载数据集

* 用户发布与接收任务

* 用户对数据集质量的评价

* 数据集报错与修改

#### 网页功能

* `/` 

  首页

* `/login`

  用户登录

  用户登陆界面有用户名、密码和验证码输入框，用户输入之后，先将用户输入的验证码与验证码的真值比较，如果相同就将用户名和密码加密后传到后台进行验证，如果验证通过则用户完成登陆。

* `/signup`

  用户注册

* `/dataset`

  展示可下载数据集

* `/upload`

  上传数据

* `/download`

  下载数据集

* `/postTask`

  发布任务

* `/task`

  查看任务
